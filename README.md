Before starting, unzip

resources folder by

right-clicking it and

hitting "extract all"

and click extract.

This should leave you

with an unzipped folder

named "resources" with

4 subdirectories.



To start containers, run
docker-compose up -d
in the same directory

as this README



Credentials for both sites:

USERNAME: admin@admin.com

PASSWORD: password



search for
http://localhost:8080/
in browser to access website



search for
http://localhost:8081/
in browser to access analytics server



WARNING:

You may need to disable your adblocker to

allow the website and the analytics server

to properly communicate. Refreshing the 

webpage for the analytics server

should reflect any new page loads that

you make on the BookStack website.
